## 🚀 HOW TO USE THIS ARCHITECTURE

### 1. HTML Structure
Use semantic HTML with class-based styling:
```html
<div class="flex items-center justify-between gap-2 mb-3">
  <h3 class="font-bold text-primary">Title</h3>
  <button class="btn btn-primary" id="myButton">Click</button>
</div>
```

### 2. Event Handling
Add event listeners in JavaScript (not HTML):
```javascript
document.getElementById('myButton')
  .addEventListener('click', handleClick);
```

### 3. Component Loading
Load CSS and JS in correct order:
```html
<!-- CSS (4 files, in order) -->
<link rel="stylesheet" href="/css/global.css" />
<link rel="stylesheet" href="/css/components.css" />
<link rel="stylesheet" href="/css/utilities.css" />
<link rel="stylesheet" href="/css/layout.css" />

<!-- JavaScript (in order of dependency) -->
<script src="/js/auth.js"></script>
<script src="/js/components/navbar.js"></script>
<script src="/js/components/footer.js"></script>
<script src="/js/main.js"></script>
```

### 4. Media Replacement
Replace placeholders when ready:
```html
<!-- Current: -->
<img src="https://placehold.co/600x400?text=Hero" />

<!-- Change to: -->
<img src="/assets/images/hero-image.jpg" />
```
